- [ ] Install Gatsby CLI
```sh
npm install -g gatsby-cli
```
- [ ] Create a new site
```sh
gatsby new gatsby-site
```

- [ ] Change directory to project folder
```sh
cd gatsby-site
```

- [ ] Start the development server
```sh
gatsby develop
```

## With npx:

- [ ] Create a new site
```sh
npx gatsby new gatsby-site
```

- [ ] Change directory to project folder
```sh
cd gatsby-site
```

- [ ] Start the development server
```sh
npx gatsby develop
```
